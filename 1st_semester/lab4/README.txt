Для массива из индивидуального задания реализовать:
1 Ввод-вывод на языке С++  (cin, cout)
2 Генераця элементов массива-случайная
3 Выполнить вторую и третью сортировку по возрастанию и убыванию (см. индивидуальный вариант)
4 Выполнить  сортировку по убыванию с помощью  стандартной функции qsort.

6 Выполнить  сортировку с помощью  стандартной функции sort ( STL: algorithm).
